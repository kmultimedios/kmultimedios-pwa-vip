## Herramientas recomendadas

## Requisitos de diseño

- Fondo: `#e63946` (rojo KMultimedios) o negro `#0a0a0a`
- El icono 192 y 512 debe tener `purpose: "maskable any"` → área segura en 80% del centro
- Formato: PNG con transparencia o fondo sólido
